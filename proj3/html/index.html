CS 143 / Project 3 / Scene Recognition with Bag of Words

This project served as an introduction to image recognition. The goal was to explore scene recognition using various methods, and to gain a better understanding of these methods through the process of comparing their respective results. The methods used were:

  1. Tiny image features grouped with k-nearest neighbor classifier

  2. Bag of words with K-nearest neighbor classifier

  3. Bag of words with Linear SVM


I also added some extensions, as you will see below.


Tiny Images and K-Nearest Neighbor

To create image features, I simply resized all of the images to a 16x16 resolution. I then ran these features on a 7-Nearest Neighbor classifier which weights votes according to their distance to each feature in the training set. This achieved 21% accuracy.


Bag of Words and K-Nearest Neighbor

Next, I created a vocabulary of visual words by densely sampling SIFT features from each training image. The SIFT features were clustered using K-means and the cluster centers were stored as words. Next, I sample SIFT features from the training and testing sets and put the features in histogram bins according which word they are closest to in feature space. The histograms are used as features for each image and sent to the nearest neighbor classifier used above. This achieved ~53% accuracy.


Bag of Words and Linear SVM

I used the same bag of words method described above and then trained a linear SVM using the image features from the training set. The following results are based on this recognition method.


Results


Depending on the number of words in the vocabulary, the percent accuracy for the bag of words and SVM implementation was around 70%. By adding spatial information using spatial pyramid matching, I was able to achieve 78% accuracy.

Spatial Pyramid Matching


In order to look at how spatial information affects scene recognition, I implemented the spatial pyramid match kernel from Lazebnik et al 2006. This implementation involved densely sampling SIFT features from each image and then splitting it up into grids of progressively finer resolution. Histograms were created for each SIFT cluster center in each grid cell at each level of resolution. Histograms were weighted according to the corresponding level of resolution and concatenated to create features to feed into the SVM. The bag of words implementation is actually analogous to this method using only one level of resolution. I used 3 levels of resolution and didn't get a chance to experiment with other levels. This would be interesting to look into.
